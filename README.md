## ✨ Возможности

*   **📯 Команда `/start` и `/help`:** Приветственное сообщение, объясняющее суть бота.
*   **ℹ️ Команда `/info`:** Краткая справка о функционале на русском языке.
*   **🔄 Полное эхо:** Бот дословно повторяет любое текстовое сообщение, которое вы ему отправите (ответом на сообщение).
